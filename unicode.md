# Unicode 



## How to use it in bash

### type directly

echo $'\xe2\x98\xa0'

echo -e '\u2620' 

echo -e '\xf0\x9f\x94\xA5'


### OSX: 

Turn on a input source called "Unicode ..." in "others" category. 

Once this keyboard is on, it is possible to hold option key and key in the unicode. e.g. 00A5
